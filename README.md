GLX and X11 sanity check

## Building
### Prerequisites
```
libx11-dev libgl-dev libglu1-mesa-dev gcc make git
```
### Cloning
```
git clone --depth=1 https://github.com/GNUfault/GLXcube.git && cd GLXcube
```
### Compiling
```
make
```
## Running
```
./main
```
## Cleaning
```
make clean
```
